## How to compile tsp into open api yaml

1. install dependency

   ```bash
   npm install
   ```

2. format the code

   ```
   make format
   ```

3. compile
   ```
   make compile
   ```

Then the output file will be in `tsp-output/schema/openapi.yaml`.
You can see the preview of the api document with vscode extention https://marketplace.visualstudio.com/items?itemName=42Crunch.vscode-openapi.
